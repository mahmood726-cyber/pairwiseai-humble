1.1 BACKGROUND AND RATIONALE
=============================

Meta-analysis is a cornerstone of evidence-based medicine, providing quantitative
synthesis of evidence from multiple studies to inform clinical decision-making,
guideline development, and healthcare policy (Egger et al., 2001; Higgins et al.,
2019). The statistical accuracy of meta-analytic calculations is critical, as
erroneous results can lead to incorrect clinical conclusions with potentially
serious implications for patient care.

Pairwise Pro version 2.2 is a web-based application designed to perform
comprehensive meta-analyses without requiring statistical programming expertise.
While such tools democratize access to meta-analytic methods, they must be
rigorously validated to ensure that the underlying calculations are mathematically
correct and numerically stable.

This validation report documents a comprehensive, independent evaluation of
Pairwise Pro v2.2 against the R metafor package (Viechtbauer, 2010), which is
widely regarded as the reference standard for meta-analysis in statistical research.
The metafor package has been extensively peer-reviewed, is cited in over 10,000
publications, and is used by regulatory agencies including the FDA and EMA.

JUSTIFICATION FOR VALIDATION
-----------------------------
1. Clinical Impact: Meta-analyses influence treatment decisions for millions of
   patients worldwide. Statistical accuracy is paramount.

2. Publication Standards: High-impact journals (e.g., Lancet, NEJM, JAMA, BMJ)
   require methodological rigor and may request validation of novel software tools.

3. Regulatory Context: Meta-analyses submitted to regulatory agencies for drug
   approvals must use validated statistical methods.

4. Scientific Integrity: Independent validation ensures that results are
   reproducible and trustworthy.

5. User Confidence: Clinicians and researchers need assurance that complex
   statistical calculations are implemented correctly.

==========================

2.3 STATISTICAL METRICS EVALUATED
==================================

For each dataset, the following metrics were computed and compared:

EFFECT SIZE METRICS
-------------------
1. Individual study effect sizes (yi)
   - Log odds ratio (OR)
   - Log risk ratio (RR)
   - Risk difference (RD)
   - Mean difference (MD)
   - Standardized mean difference (SMD/Hedges' g)
   - Log hazard ratio (HR)
   - Logit proportion (single-arm)

2. Individual study variances (vi)
   - Calculated using appropriate formula for each measure
   - Incorporating continuity corrections where applicable

3. Individual study standard errors (sei)
   - sei = sqrt(vi)

HETEROGENEITY METRICS
---------------------
1. Q statistic
   - Cochran's Q test for heterogeneity
   - Chi-square distribution with df = k - 1
   - P-value for test of homogeneity

2. I² statistic
   - Percentage of variability due to heterogeneity
   - I² = max(0, 100% × (Q - df) / Q)
   - Interpretation: 0-40% low, 30-60% moderate, 50-90% substantial, 75-100% considerable

3. H² statistic
   - Ratio of total to sampling variability
   - H² = Q / df
   - H² = 1 indicates no heterogeneity

4. tau² (tau-squared)
   - Variance of true effects across studies
   - Estimated using 8 different methods (see Section 5)

5. tau
   - Standard deviation of true effects
   - tau = sqrt(tau²)

POOLED EFFECT METRICS
---------------------
1. Pooled effect estimate (theta, θ)
   - Random-effects weighted average
   - Weights: wi* = 1/(vi + tau²)
   - Formula: θ = Σ(wi* × yi) / Σwi*

2. Standard error of pooled estimate
   - SE(θ) = sqrt(1 / Σwi*)

3. 95% Confidence interval
   - Wald method: θ ± 1.96 × SE(θ)
   - HKSJ method: θ ± t(k-2) × SE_HKSJ

4. Test statistic
   - z = θ / SE(θ)  [Wald]
   - t = θ / SE_HKSJ  [HKSJ]

5. P-value
   - Two-sided test of null hypothesis
   - Normal approximation or t-distribution

6. Exponentiated values
   - For ratio measures: exp(θ), exp(CI_lower), exp(CI_upper)

PREDICTION INTERVAL METRICS
---------------------------
1. Standard prediction interval
   - 95% PI for true effect in a new study
   - PI = θ ± t(k-2) × sqrt(tau² + SE²)

2. Noma prediction interval (with tau² uncertainty)

3. Conformal prediction interval

PUBLICATION BIAS METRICS
------------------------
1. Egger's regression test
   - Linear regression of effect size on standard error
   - Test of intercept ≠ 0
   - P-value for asymmetry

2. Peters' test (for binary data)
   - Alternative to Egger's for OR/RR
   - Less affected by heterogeneity

3. Trim and Fill
   - Duval and Tweedie method
   - Number of imputed studies (k0)
   - Side of asymmetry (left/right)
   - Adjusted pooled estimate

SENSITIVITY ANALYSIS METRICS
-----------------------------
1. Leave-one-out analysis
   - Pooled estimate omitting each study in turn
   - Influence of individual studies

2. Baujat plot coordinates
   - Overall heterogeneity contribution
   - Influence on pooled estimate

3. Cumulative meta-analysis
   - Sequential addition of studies

3. Non-convergence of iterative methods in standard scenarios

6. Systematic bias detected across multiple datasets

  a = events in treatment arm
  b = non-events in treatment arm (n_t - a)
  c = events in control arm
  d = non-events in control arm (n_c - c)

Test Case:
  Input: a=20, b=80, c=40, d=60

Results:
  R metafor logOR:        -0.980829253
  Pairwise Pro logOR:     -0.980829253
  Absolute difference:     0.000000000

  R metafor var(logOR):    0.104166667
  Pairwise Pro var(logOR): 0.104166667
  Absolute difference:     0.000000000

Status: ✓ PASSED - PERFECT MATCH

Clinical interpretation: The formula correctly computes that treatment reduces

TEST 2: CONTINUITY CORRECTION FOR ZERO EVENTS
----------------------------------------------
Formula (when zero events present):
  Add constant c to all cells:
  a_cc = a + c
  b_cc = b + c
  c_cc = c + c
  d_cc = d + c

Standard approach: c = 0.5

Test Case:
  Input: a=0, b=50, c=5, d=45
  Continuity correction: 0.5

Results:
  R metafor logOR:        -2.502156275

TEST 9: DERSIMONIANLAIRD (DL) TAU²
-------------------------------------
Formula:
  wi = 1/vi  [fixed-effect weights]
  theta_FE = Σ(wi × yi) / Σwi
